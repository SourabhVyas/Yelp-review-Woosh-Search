# Yelp-review-Woosh-Search


> This project applies Natural Language Processing (NLP) and Information Retrieval (IR) techniques to analyze Yelp business reviews, focusing on businesses located in Santa Barbara, California.

---

### Features
- **Data preprocessing**: tokenization, stopword removal, stemming (Porter, Lancaster).  
- **Writing style analysis**: heuristic features (capitalization, pronouns, readability, punctuation).  
- **Search engine**: Whoosh-based keyword search and TF‑IDF + cosine similarity for similar-review retrieval.  
- **Trend analysis**: VADER sentiment, TF‑IDF time series, NMF topic modeling, aspect-based analysis.  
> The study encompasses four major components: data preprocessing and tokenization, writing style analysis, search engine development, and semantic and sentiment trend analysis.
